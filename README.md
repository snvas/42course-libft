# Libft

## Overview
Uma reimplementação das bibliotecas padrão da linguagem C (standard C Library function) 
criadas com as regras da Norminette e deve se comportar como as originais.

## Não Usar

- For
- Do... While
- Switch
- Case
- Goto
- Variable Length Array (VLA)
